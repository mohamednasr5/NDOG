📚 How to Use a Block Explorer

A block explorer is a search engine for blockchain transactions. Here's how to use one:

  1. Search by address — Paste a wallet address to see its balance and history.
  2. Search by transaction hash — Each transaction has a unique ID you can look up to verify it was confirmed.
  3. Search by block number — See all transactions included in a specific block.
  4. Filter by type — Use the dropdown above to narrow by credit, debit, claim, stake, mission, or referral.
  5. Verify timestamps — Each transaction shows when it was recorded on-chain.

The NileDogs explorer mirrors the on-chain state from Firebase Realtime Database. After mainnet launch, all entries will be anchored to a public blockchain.
